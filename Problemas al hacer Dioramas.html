Problemas al hacer Dioramas

PROBLEMA CAUSA SOLUCIÓN PREVENCIÓN
PROBLEMAS DE ESTRUCTURA Y BASE
Base inestable o que se tambalea Material inadecuado (cartón fino), grosor insuficiente, construcción deficiente, apoyo desigual Reforzar con listones de madera debajo, cambiar a base más rígida (MDF, contrachapado), nivelar superficie Usar bases de al menos 15mm de grosor, probar estabilidad antes de continuar, usar nivelador
Deformación de la base por combadura Humedad, peso excesivo concentrado, material de baja calidad, cambios de temperatura Secar completamente, reforzar con estructura interna, redistribuir peso, prensar con peso Sellar la base con imprimación, calcular peso máximo, usar soportes internos cruzados
Grietas en la base Expansión/contracción del material, impactos, sobrecarga puntual, material defectuoso Reparar con masilla para madera, reforzar zona con fibra de vidrio, cambiar sección dañada Usar maderas estables, evitar cambios bruscos de humedad, distribuir cargas
Elementos se caen o despegan Adhesivo inadecuado, superficie mal preparada, peso excesivo, vibración, temperatura Limpiar superficies con alcohol, usar adhesivo estructural, añadir fijaciones mecánicas Rugosidad superficie con lija, usar adhesivos específicos, probar resistencia antes
Juntas entre secciones visibles Cortes imprecisos, diferencia de altura, falta de tratamiento de juntas Usar masilla de carrocero, lijar a ras, texturizar para disimular Medir y cortar con precisión, usar herramientas guiadas, planificar juntas
Hundimientos o desniveles inesperados Terreno no planificado, falta de soporte interno, material blando Rellenar con espuma expansiva, añadir soportes ocultos, remodelar zona Hacer maqueta previa, calcular estructura interna, usar materiales adecuados
PROBLEMAS DE PINTURA Y ACABADOS
Pintura se cuartea o descascara Superficie mal preparada, imprimación inadecuada, capas muy gruesas, incompatibilidad Lijar totalmente, limpiar con desengrasante, aplicar imprimación específica y repintar Siempre usar imprimación compatible, limpiar superficie, aplicar capas de 0.1mm máximo
Colores desiguales o manchados Mezcla insuficiente, aplicación irregular, contaminación del pincel, secado parcial Lijar suavemente grano 400, limpiar y repintar con técnica uniforme, usar rodillo Mezclar 5 minutos mínimo, usar pinceles nuevos, mantener borde húmedo
Pintura no seca correctamente (pegajosa) Humedad alta (+70%), temperatura baja (-15°C), capa gruesa, producto caducado Mejorar ventilación, usar deshumidificador, aplicar calor 30°C, esperar más tiempo Controlar humedad 40-60%, temperatura 18-25°C, verificar fechas
Marcas de pincel muy visibles Pincel duro/gastado, técnica incorrecta, pintura muy espesa, secado rápido Lijar con grano 600, usar pincel pelo natural, diluir 10% la pintura Usar pinceles calidad, técnica cruzada, añadir retardante secado
Diferencias de brillo en la superficie Absorción desigual, aplicación irregular, mezcla de tipos de pintura Aplicar capa selladora uniforme, lijar suave y reaplicar acabado Sellar superficie porosa, usar misma marca y línea de producto
Pintura se ampolla o burbujea Humedad en la superficie, calor excesivo durante aplicación, superficie contaminada Eliminar ampollas, secar superficie 24h, lijar y repintar Asegurar superficie seca, evitar sol directo durante pintura
Sangrado de colores subyacentes Imprimación insuficiente, solventes agresivos, pigmentos migrantes Aplicar sellador específico, usar pintura base agua sobre problemáticas Usar imprimación bloqueadora, probar compatibilidad en muestra
PROBLEMAS DE MATERIALES Y TEXTURAS
Texturas artificiales poco realistas Un solo material, falta variación, escala incorrecta, patrones repetitivos Mezclar 3-4 materiales diferentes, variar tamaños, estudiar fotos reales Investigar texturas reales, usar materiales naturales, crear muestras previas
Césped artificial se ve falso Un solo tono verde, altura uniforme, distribución regular, brillo excesivo Pintar con 3 tonos verdes, cortar alturas variables, añadir tierra Usar múltiples tonos, aplicar irregularmente, matizar con pintura
Materiales se deshacen o desintegran Baja calidad, falta de fijador, manipulación excesiva, humedad Aplicar fijador acrílico en spray, reemplazar secciones dañadas Usar materiales archivo, aplicar sellador preventivo, mínima manipulación
Arena o gravilla se dispersa Fijación insuficiente, vibración, manipulación, corrientes aire Aplicar cola blanca diluida con spray, aspirar exceso suelto Fijar con cola antes del secado, aplicar por zonas, usar vitrina
Espuma de poliuretano se deforma Calor excesivo, presión continuada, calidad baja, envejecimiento Sustituir sección deformada, reforzar con núcleo rígido interno Usar espuma alta densidad, evitar temperaturas +30°C, núcleo rígido
Musgos y líquenes naturales se deterioran Humedad excesiva, luz UV, manipulación, insectos Sustituir por preservados, aplicar conservante natural Usar musgos preservados, controlar humedad, filtrar UV
Agua artificial se ve poco convincente Superficie plana, falta transparencia, color uniforme, sin movimiento Usar resina transparente, crear ondas con calor, añadir profundidad color Estudiar agua real, usar resinas específicas, crear diferentes profundidades
Materiales metálicos se oxidan Humedad ambiente, falta protección, calidad baja del metal Eliminar óxido con convertidor, aplicar imprimación antioxidante Usar metales nobles o tratados, aplicar protección preventiva
PROBLEMAS DE ESCALA Y PROPORCIÓN
Elementos fuera de escala Falta planificación, no usar referencias, estimación visual incorrecta Recalcular con regla de tres, reemplazar elementos incorrectos Definir escala numérica (1:50), usar escalímetro, hacer tabla conversión
Perspectiva forzada incorrecta Colocación errónea, ángulo visual incorrecto, falta conocimiento perspectiva Reposicionar según punto de fuga, ajustar tamaños progresivamente Estudiar perspectiva cónica, usar líneas guía, definir punto observación
Figuras humanas desproporcionadas Mezcla de escalas diferentes, no considerar edad/sexo, error fabricante Modificar físicamente figuras, crear bases compensatorias Comprar misma marca/serie, medir alturas reales, usar referencias
Edificios no respetan proporciones arquitectónicas Simplificación excesiva, falta referencias técnicas, construcción amateur Consultar planos reales, ajustar proporciones principales Usar planos arquitectónicos, respetar cánones constructivos
Vegetación fuera de escala natural No considerar crecimiento real plantas, usar materiales inadecuados Investigar tamaños reales especies, seleccionar según escala Consultar guías botánicas, hacer tabla especies/tamaños
PROBLEMAS DE ILUMINACIÓN
Iluminación desigual o sombras incorrectas Múltiples fuentes sin coordinación, posición inadecuada, potencia desigual Unificar dirección luz principal, usar reflectores para sombras Planificar con una fuente principal + relleno, usar dimmer
Cables o elementos eléctricos visibles Falta planificación instalación, cables muy gruesos, recorrido inadecuado Crear canales ocultos, usar cables planos, pintar del color base Planificar instalación eléctrica antes decorar, usar cable fino
LEDs se ven como puntos de luz Falta difusión, LEDs muy potentes, distancia insuficiente Añadir material difusor, aumentar distancia, reducir intensidad Usar tiras LED, material difusor desde diseño, calcular distancias
Calor excesivo de la iluminación Lámparas incandescentes, ventilación insuficiente, potencia excesiva Cambiar a LED, mejorar ventilación, usar temporizador Usar solo LED, calcular disipación térmica, ventilación adecuada
Color de luz inadecuado para la escena Temperatura color incorrecta, no considerar ambiente, mezcla fuentes Usar filtros de color, cambiar tipo de LED, unificar fuentes Elegir temperatura según ambiente (3000K cálido, 6000K frío)
Reflejos no deseados en superficies Superficies demasiado brillantes, ángulo luz incorrecto Matizar superficies brillantes, cambiar ángulo iluminación Usar acabados mate/satinado, probar ángulos iluminación
PROBLEMAS DE CONSERVACIÓN Y MANTENIMIENTO
Acumulación de polvo generalizada Falta protección, materiales electroestáticos, ventilación deficiente Limpiar con pincel natural suave, aspiradora boquilla, paño antiestático Crear vitrina hermética, usar materiales antiestáticos, filtrar aire
Decoloración por luz solar directa Rayos UV, exposición prolongada, materiales sensibles, calor Reubicar zona sin sol directo, usar filtros UV en cristales Materiales resistentes UV, filtros ventanas, rotación ocasional
Daños por humedad (moho, deformación) Humedad >65%, ventilación deficiente, materiales orgánicos Deshumidificador, mejorar ventilación, tratamiento antifúngico Mantener 45-55% humedad, higrómetro, materiales tratados
Insectos que dañan materiales orgánicos Materiales naturales sin tratar, ambiente propicio, falta limpieza Tratamiento insecticida específico, eliminar zonas afectadas Tratar materiales orgánicos, limpieza regular, control ambiental
Desgaste por manipulación excesiva Transporte frecuente, falta protección, materiales delicados Reforzar zonas desgastadas, limitar manipulación Diseñar zonas de agarre, usar materiales resistentes, funda protección
Cambios dimensionales por temperatura Variaciones térmicas bruscas, materiales diferentes, falta compensación Permitir acomodación gradual, ajustar uniones Controlar temperatura ambiente, usar juntas flexibles
PROBLEMAS DE TRANSPORTE Y ALMACENAMIENTO
Roturas durante transporte Embalaje insuficiente, elementos frágiles, vibraciones, impactos Reparar con técnicas apropiadas, reforzar para futuro Embalaje profesional, elementos desmontables, amortiguación adecuada
Desplazamiento de elementos móviles Fijación temporal insuficiente, vibración, aceleración Reposicionar elementos, mejorar fijación temporal Sistema fijación temporal, documentar posiciones originales
Deformación por almacenamiento inadecuado Posición incorrecta, peso concentrado, temperatura, humedad Restaurar forma con calor controlado, reforzar estructura Almacenar horizontal, soporte uniforme, ambiente controlado
Deterioro por almacenamiento prolongado Falta mantenimiento, condiciones inadecuadas, materiales degradables Programa restauración, sustitución elementos dañados Revisiones periódicas, condiciones museo, documentación estado
PROBLEMAS DE TIEMPO Y PLANIFICACIÓN
No cumplir plazos de entrega Subestimar tiempo, complicaciones imprevistas, perfeccionismo excesivo Priorizar elementos esenciales, aumentar horas trabajo, pedir prórroga Cronograma realista +30% tiempo, hitos intermedios, plan contingencia
Agotar materiales a medio proyecto Cálculo incorrecto, no considerar desperdicios/errores, cambios diseño Buscar equivalentes, adaptar diseño, comprar urgente Calcular +25% extra, lista completa previa, proveedores alternativos
Cambios de diseño a medio proyecto Planificación insuficiente, nuevas ideas, problemas técnicos Evaluar impacto tiempo/coste, implementar solo críticos Diseño detallado previo, bocetos múltiples, congelar diseño
Fatiga y pérdida de motivación Proyecto muy largo, trabajo intensivo, falta descansos Planificar descansos, dividir en fases, buscar ayuda Objetivos intermedios, trabajo en sesiones, recompensas parciales
PROBLEMAS TÉCNICOS ESPECÍFICOS
Burbujas en resina epoxi o barniz Mezcla muy rápida, temperatura baja, humedad, contaminación Eliminar con pistola calor suave, lijar y reaplicar Mezclar lentamente, temperatura 20-25°C, deshumidificar, limpiar superficie
Uniones visibles entre piezas ensambladas Cortes imprecisos, ensamblaje deficiente, diferente expansión materiales Masilla específica material, lijar progresivo, repintar zona Herramientas precisión, ingletes perfectos, probar ajuste en seco
Figuras no mantienen posición estable Centro gravedad alto, base pequeña, peso mal distribuido Ampliar base, añadir peso interno, modificar postura Calcular centro gravedad, base mínima 3x altura, peso en base
Adhesivos no curan correctamente Temperatura baja, humedad incorrecta, proporción mal medida, caducidad Aplicar calor suave, eliminar y reaplicar correctamente Verificar condiciones y fechas, medir con precisión, probar antes
Sonidos no deseados (crujidos, vibraciones) Uniones rígidas, resonancia, falta amortiguación Añadir material amortiguante, reforzar uniones flojas Diseñar uniones flexibles, usar materiales amortiguantes
Problemas con motores y mecanismos Sobrecarga, falta lubricación, desalineación, desgaste Revisar mecánica, lubricar, realinear ejes, cambiar piezas Cálculo cargas, mantenimiento preventivo, componentes calidad
PROBLEMAS ESTÉTICOS Y DE COMPOSICIÓN
Composición desequilibrada visualmente Falta planificación compositiva, elementos mal distribuidos, sin punto focal Reposicionar elementos principales, crear jerarquía visual Estudiar regla tercios, crear bocetos compositivos, definir punto focal
Colores no armonizan entre sí Falta teoría color, colores saturados excesivos, sin unidad cromática Aplicar filtros color unificadores, reducir saturación, crear paleta coherente Estudiar círculo cromático, definir paleta previa, usar colores análogos
Falta de profundidad visual Todos elementos mismo plano, falta gradación tonal, sin superposiciones Crear diferentes planos, graduar tonos según distancia, añadir superposiciones Planificar en 3 planos (primer, medio, fondo), usar perspectiva aérea
Exceso de detalles que confunden Querer mostrar todo, falta jerarquización, elementos compiten atención Simplificar zonas secundarias, destacar elemento principal Definir qué es principal/secundario, usar detalle progresivo
Estilo incoherente entre elementos Mezcla estilos, diferentes niveles detalle, épocas incompatibles Unificar tratamiento, adaptar elementos discordantes Definir estilo único, crear guía visual, verificar coherencia
PROBLEMAS DE SEGURIDAD
Elementos cortantes o punzantes expuestos Materiales sin proteger, bordes afilados, puntas metálicas Cubrir con protectores, limar bordes, añadir señalización Evaluar riesgos desde diseño, usar materiales seguros, proteger bordes
Inestabilidad que puede causar caídas Centro gravedad alto, base insuficiente, apoyo precario Ampliar base, añadir contrapesos, fijar a superficie Calcular estabilidad, probar con empuje lateral, base segura
Materiales tóxicos o alergénicos Uso productos químicos, materiales sin verificar, falta ventilación Sellar materiales problemáticos, mejorar ventilación Verificar toxicidad materiales, usar EPIs, etiquetado seguridad
Riesgo eléctrico por instalación deficiente Cables pelados, conexiones incorrectas, falta aislamiento Revisar instalación, aislar correctamente, usar protecciones Instalación por profesional, usar voltajes seguros, protecciones adecuadas
PROBLEMAS DE PRESUPUESTO Y RECURSOS
Sobrecosto del proyecto Estimación incorrecta, cambios diseño, materiales caros, desperdicios Buscar alternativas económicas, simplificar diseño, reutilizar materiales Presupuesto detallado +20%, comparar proveedores, lista prioridades
Herramientas inadecuadas o insuficientes No evaluar necesidades, presupuesto limitado, herramientas inadecuadas Alquilar herramientas específicas, adaptar técnicas, buscar préstamos Lista herramientas necesarias, evaluar alquiler vs compra
Desperdicio excesivo de materiales Falta planificación cortes, errores repetidos, compras excesivas Planificar uso restos, proyectos secundarios con sobrantes Planos de corte optimizados, calcular exacto, guardar restos útiles
PROBLEMAS AMBIENTALES Y ECOLÓGICOS
Uso materiales no sostenibles Falta conciencia ambiental, desconocimiento alternativas, coste Sustituir por eco-materiales cuando sea posible Investigar materiales sostenibles, proveedores responsables
Generación excesiva de residuos Falta planificación, materiales no reutilizables, embalajes excesivos Separar para reciclaje, buscar usos alternativos Planificar gestión residuos, materiales reutilizables, compra a granel
Emisiones tóxicas durante construcción Adhesivos con solventes, pinturas base solvente, falta ventilación Mejorar ventilación inmediatamente, usar mascarillas Productos base agua, ventilación adecuada, EPIs apropiados
PROBLEMAS DE DOCUMENTACIÓN Y REGISTRO
Pérdida de información del proceso Falta documentación, no fotografiar etapas, memoria deficiente Intentar reconstruir información, consultar referencias Fotografiar cada etapa, diario del proyecto, archivo organizado
Dificultad para reproducir técnicas exitosas No documentar procedimientos, falta detalle técnico Experimentar hasta recuperar técnica, consultar expertos Documentar recetas y procedimientos, vídeos proceso
Falta de respaldo ante daños No documentar estado original, falta fotografías detalle Fotografiar estado actual como nueva referencia Fotografías alta resolución todos los ángulos, medidas exactas
PROBLEMAS SOCIALES Y DE COMUNICACIÓN
Malentendidos con el cliente/profesor Especificaciones poco claras, falta comunicación, expectativas diferentes Reunión aclaratoria, documentar acuerdos, mostrar avances Brief detallado por escrito, bocetos aprobados, comunicación regular
Críticas negativas inesperadas No validar concepto, falta investigación, subjetividad Analizar críticas constructivas, defender decisiones técnicas Presentar bocetos previos, investigar referencias, justificar decisiones
Falta reconocimiento del trabajo realizado No documentar esfuerzo, trabajo invisible, falta comunicación valor Crear portafolio detallado, documentar proceso completo Documentar tiempo invertido, explicar complejidad técnica
PROBLEMAS TÉCNICOS DE HERRAMIENTAS
Herramientas de corte desafiladas Uso intensivo, falta mantenimiento, calidad baja Afilar profesionalmente, sustituir hojas, ajustar técnica Mantenimiento regular, herramientas calidad, uso apropiado
Herramientas eléctricas fallan Sobrecarga, falta mantenimiento, desgaste, mala calidad Reparar si es viable, alquilar/comprar alternativa Mantenimiento preventivo, no sobrecargar, calidad adecuada
Mediciones imprecisas repetidas Herramientas descalibradas, técnica incorrecta, instrumentos inadecuados Calibrar instrumentos, verificar con múltiples medidas Instrumentos precisión adecuada, calibración regular, técnica correcta
PROBLEMAS DE ACABADO FINAL
Resultado final no coincide con boceto Cambios durante proceso, limitaciones técnicas, boceto poco realista Evaluar si mejorar resultado o aceptar limitaciones Bocetos realistas técnicamente, maquetas prueba, validar factibilidad
Calidad desigual entre diferentes zonas Fatiga progresiva, aprendizaje durante proceso, cambio materiales Homogeneizar calidad, rehacer zonas deficientes Mantener estándares constantes, descansos regulares, materiales uniformes
Detalles finales no se aprecian Escala muy pequeña, ubicación inadecuada, falta contraste Realzar con iluminación, aumentar contraste, reposicionar Evaluar visibilidad desde distancia normal, jerarquizar detalles
